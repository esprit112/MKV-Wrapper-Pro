# Contributing to MKV Wrapper Pro

We welcome contributions! To keep the code quality high and the "Pro" aesthetic consistent, please follow these guidelines:

1.  **Fork the Project.**
2.  **Create your Feature Branch** (`git checkout -b feature/AmazingFeature`).
3.  **Commit your Changes** (`git commit -m 'Add some AmazingFeature'`).
4.  **Push to the Branch** (`git push origin feature/AmazingFeature`).
5.  **Open a Pull Request.**

### Code Standards
- Use functional React components.
- Maintain the Atomic CSS (Tailwind) approach.
- Ensure any hardware bridge changes are cross-platform compatible where possible.
